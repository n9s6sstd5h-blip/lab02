## Structure

- `include/` - header files.
- `sources/` - source files.
- `examples/` - example programs.

## Build Examples

```sh
clang++ -std=c++17 -Iinclude sources/print.cpp examples/example1.cpp -o example1
clang++ -std=c++17 -Iinclude sources/print.cpp examples/example2.cpp -o example2
```
